# Contributing to Awesome Claude Code Plugins

Thanks for your interest in contributing to this awesome list!

## Adding New Plugins

1. Check the existing format in README.md
2. Add your plugin in the appropriate category
3. Include: name, description, URL, and tags

## Guidelines

- Only add working, maintained plugins
- Use consistent formatting
- Test links before submitting

## Submitting

1. Fork this repo
2. Add your changes
3. Create a Pull Request

## Bounty Program

Earn RTC for contributions! See: https://github.com/Scottcjn/rustchain-bounties
